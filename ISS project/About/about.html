About

This is a 2025 summer project designed by Parth Raval, a current Electrical Engineering student at the University of Scranton.

This project uses API calls to display various real-time telemetry points for the ISS. It also uses chart.JS to graph some of these data points.

Additionally, this project allows the user to perform calculations related to the ISS and interact with other space-based widgets.


This project was inspired by NASA OpenMCT software. I hope to continue to better this project and include more advance features in the future.


Project was built with HTML, CSS, and Javascript.
